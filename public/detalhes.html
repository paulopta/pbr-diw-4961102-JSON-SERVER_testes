1ª Etapa

Busque a base de dados "livros.js" (assets/db/livros.json) no servidor JSON SERVER por requisição AJAX no arquivo "detalhes.js" e convertendo o resultado em JSON e armazenando em uma variável "livros";.

Crie uma variável e atribua a ela a função new URLSeacrchParams(Location.search). Esta função atribui a variável criada os valores que estão nos parâmetros da URL.

Utilizando a variável criada anteriormente, atribua a ela o método .get('id') para receber o valor do atributo id que está na URL. Crie uma variável id para receber essa sentença e armazenar o valor.

Utilize a função .find(function (elem) { return elem.id==id}); no objeto JSON "livros" que foi criado. Essa função busca um valor e o retorna uma linha do objeto se ele for verdadeiro, nesse caso a chave de busca é o id. Atribua uma variável a essa sentença.

2ª Etapa

Crie uma sentença para verificar se a variável criada anteriormente é verdadeira ou falsa.

Caso ela seha verdadeira, ela se tornou um objeto contendo todos os atributos da linha pesquisada no objeto JSON livros.

Para acessar os valores acrescenter um . a variável e escreva o atributo indicado no JSON, como por exemplo ".titulo". Utilize a função .innerHTML para alterar os valores dos elementos span ("id","titulo","autor" e "ano") nessa página.

Caso não exista id pesquisado, utilize a função .innerHTML para alterar os valores dos elementos span ("id","titulo","autor" e "ano") com os dizeres "não disponível".
